# 当前对话需求说明

## 协作边界

- 本文件仅记录当前对话下确认的需求。
- 同一项目目录下，不同对话之间若没有明确需求指引，默认互不相关。
- 后续新会话不得因为本对话内容推翻、重写其他会话下已经产出的文件。
- 若新会话提出新增需求，应优先新增页面或输出到其他文件；除非用户明确要求修改既有文件，否则不进行代码重写。

## 页面目标

构建一个动态项目排期工具，用于单个 SKU 下多项目大类、多物料任务并行排期。

页面应支持：

- 项目基础信息填写。
- 项目大类选择、增删、重命名。
- 不同项目大类设置不同启动时间。
- 根据模板节点自动计算各项目大类完成时间。
- 自动生成日历排期。
- 支持导出 PNG 图片。

## 基础信息

基础设置区包含：

- 时间表标题。
- 项目名称。
- 自动交付日期。
- 展示起始周。
- 展示周数。
- 休息规则。
- 假期开始。
- 假期结束。

自动交付日期规则：

- 不由用户手动填写。
- 自动显示为当前已选项目大类中的最晚完成时间。

## 休息日与假期

日历规则支持：

- 无休。
- 单休：周日不排期。
- 双休：周六、周日不排期。
- 假期范围：假期开始到假期结束之间不排期。

显示规则：

- 单休/双休日期不进行排期，日历任务区域为空。
- 假期日期不进行排期，并在日历中显示“假期”。
- 完成时间需要根据休息日和假期自动顺延。
- 导出图片需要同步体现休息日和假期规则。

## 项目大类

项目大类支持：

- 勾选启用。
- 自由新增。
- 删除。
- 重命名。
- 自定义日历显示颜色。
- 设置启动时间。
- 自动计算完成时间。
- 进入自定义模板二级页面。

默认项目大类包括：

- AMAZON&NPC图片项目。
- AMAZON视频项目。
- TK图片项目。
- TK视频项目。

颜色规则：

- 每个项目大类有独立颜色。
- 颜色同步影响项目大类行、日历任务色块、图例和导出图片。

## 共用环节

多项目大类共用一组前置环节。

默认共用环节：

- 方案制定。
- 方案会议。

共用环节要求：

- 只生成一组。
- 所有已选项目大类共用。
- 支持进入“共用环节模板”二级页面自定义。
- 支持修改节点名称、节点时间、重点样式。
- 支持新增、删除节点。
- 支持拖拽调整节点顺序。

## 项目模板

每个项目大类都有独立模板。

默认模板：

AMAZON&NPC图片项目：

- 3D/AI图片制作。
- 图片审核修改。
- 平面排版。
- 平面审核修改。
- 图片定稿。

AMAZON视频项目：

- 脚本修改。
- 脚本定稿。
- 视频拍摄/分镜制作。
- 视频生成。
- 视频剪辑。
- ACOPY。
- 审核修改。
- BCOPY。

TK图片项目：

- 图片制作。
- 平面排版。
- 审核修改。
- 图片交付。

TK视频项目：

- 脚本制作。
- 视频拍摄。
- 视频剪辑。
- ACOPY。
- 审核修改。
- BCOPY。

模板要求：

- 支持进入二级页面编辑。
- 支持节点名称自定义。
- 支持节点时间按天数自定义。
- 支持重点样式设置。
- 支持新增、删除节点。
- 支持节点顺序自由拖拽调整。

## 日历显示

日历要求：

- 同一日期支持多个阶段并行显示。
- 同一日期下多个项目大类任务以堆叠色块显示。
- 项目大类图例显示在预览表项目大类栏中。
- 图例需包含项目大类名称和对应色块。
- 日历色块颜色来源于项目大类自定义颜色。
- 假期显示为“假期”。
- 单休/双休休息日任务区域为空。

## 导出图片

导出要求：

- 点击“导出当前项目图片”后生成 PNG。
- 由于部分预览浏览器不支持直接下载，导出后应先显示图片预览弹窗。
- 弹窗包含：
  - 下载 PNG。
  - 打开图片。
  - 关闭。
- 导出图片应同步包含：
  - 自动交付日期。
  - 项目大类图例。
  - 日历任务色块。
  - 休息日/假期排期规则。
