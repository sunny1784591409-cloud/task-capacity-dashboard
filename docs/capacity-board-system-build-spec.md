# 摄影摄像产能看板系统建设说明

本文档用于把当前“摄影摄像产能看板”完整交接给其他设备上的 Codex。目标是让新的 Codex 能根据本文档重新建设出同等功能的静态 Web 系统。

## 一、系统定位

建设一个桌面端优先的摄影/摄像产能看板，用于管理导入任务、手动临时任务、自动排期、个人排期日历、月度 SKU 产能统计和已完成临时任务数据库。

系统当前为静态前端项目，无后端服务，数据保存在浏览器 `localStorage` 中。推荐技术栈：

- `index.html`
- `styles.css`
- `script.js`
- Excel/CSV 导入可使用 `xlsx@0.18.5` CDN
- 本地预览可使用任意静态文件服务器

当前项目入口建议为：

```text
projects/capacity-board/index.html
projects/capacity-board/styles.css
projects/capacity-board/script.js
```

## 二、页面结构

系统主导航包含三个一级页面：

1. 摄影任务后台
2. 摄像任务后台
3. 产能看板

摄影任务后台和摄像任务后台只保留任务管理能力，不再显示后台日历看板。日历排期统一放在产能看板里的个人日历视图中。

### 摄影任务后台

包含以下模块：

- Excel/CSV 文件导入
- 粘贴 Excel 表格导入
- 下载导入模板
- 手动填写临时任务
- 导入任务时长规则配置
- 摄影导入任务列表
- 摄影临时任务列表
- 查看已完成临时任务入口

### 摄像任务后台

包含以下模块：

- Excel/CSV 文件导入
- 粘贴 Excel 表格导入
- 下载导入模板
- 手动填写临时任务
- 导入任务时长规则配置
- 摄像导入任务列表
- 摄像临时任务列表
- 查看已完成临时任务入口

### 产能看板

包含以下模块：

- 顶部组产能汇总
  - 摄影组剩余 SKU
  - 摄像组剩余 SKU
- 月份切换
  - 本月
  - 次月
- 摄影个人 SKU 产能卡片
- 摄像个人 SKU 产能卡片
- 点击个人卡片进入个人任务排期日历
- 个人日历支持查看上月、本月、次月
- 个人日历支持拖拽任务调整排期
- 指标和排期规则说明使用问号图标悬停展示

### 导入字段自动识别

Excel/CSV/粘贴导入时，需要自动识别相关字段后导入，不要求表头完全等于模板字段。

字段识别规则：

- 表头识别时忽略大小写、空格、括号、冒号、下划线、短横线和常见标点。
- `sku别名` 可识别：`sku`、`SKU编码`、`SKU编号`、`SKU名称`、`款号`、`商品编码`、`产品编码` 等。
- `产品标题` 可识别：`商品标题`、`产品名称`、`商品名称`、`品名`、`标题`、`名称` 等。
- `产品类型` 可识别：`商品类型`、`产品分类`、`商品分类`、`品类`、`类目` 等。
- `最早交期` 可识别：`最早交付时间`、`最早交货期`、`最早完成时间`、`交期`、`交付时间` 等。
- 摄像导入字段需要支持常见同义表头：
  - `任务语种` 可识别：`语种`、`语言`、`视频语种`、`任务语言`、`站点`、`国家`。
  - `任务类型` 可识别：`视频类型`、`拍摄类型`、`内容类型`、`视频内容`、`视频需求`、`摄像内容`。
  - `任务状态` 可识别：`拍摄状态`、`分配状态`、`状态`、`进度`。
  - `优先级` 可识别：`优先`、`优先程度`、`紧急程度`、`等级`。
  - `创建时间` 可识别：`创建日期`、`提交时间`、`提交日期`、`下单时间`、`需求创建时间`。
  - `关联sku` 可识别：`关联SKU`、`关联sku别名`、`关联产品`、`关联款号`、`主sku`、`父sku`。
  - `时长` 可识别：`任务时长`、`预计时长`、`预计耗时`、`预估耗时`、`耗时`、`天数`、`工期`，并支持 `2天`、`1.5 天` 这类文本。
- 其他字段也需要支持常见同义表头，如 `拍摄人/负责人/执行人`、`到港日期/预计到港`、`拍摄类型/内容类型`、`创建日期/创建时间`。

## 五、手动填写临时任务

摄影和摄像的临时任务字段一致：

```text
sku别名
对接人
任务备注
摄影师/摄像师
预计耗时
预计完成时间
```

规则：

- 手动填写的任务为临时任务。
- 临时任务有单独任务列表。
- 临时任务和导入任务一起参与排期。
- 临时任务旧任务不受 Excel/粘贴重新导入影响。
- 摄影师/摄像师选项为单选：
  - 吕皇勇
  - 汤崇武
- 预计耗时最小单位为 `0.5天`。
- 添加临时任务后立即自动排期。
- 已有排期的临时任务在重新导入任务时保持原排期，不被自动改变。

## 六、默认时长规则

导入任务根据“拍摄内容/任务类型”匹配预计时长。

默认规则：

```text
AMAZON出图：2个工作日
AMAZON&NPC出图：3个工作日
安装视频：2个工作日
场景视频：2个工作日
AI视频：4个工作日
重点项目：5个工作日
```

要求：

- 支持新增、删除、修改规则。
- 摄影按拍摄内容匹配。
- 摄像按任务类型匹配。
- 摄像导入任务如导入表中提供“时长”，以导入时长为准，并标记为手动时长。
- 摄像导入任务如未提供“时长”，通过识别任务类型自动填充时长。
- 修改摄像任务类型、时长、状态后，需要自动刷新排期。

## 七、通用排期规则

通用规则：

- 默认双休，周六周日不排期。
- 排期从当前日期之后的最近工作日开始。
- 支持 `0.5天` 任务。
- 自动排期不需要按钮触发。
- 导入任务或新增临时任务后自动刷新排期。
- 删除任务、修改任务字段、修改时长规则后自动刷新排期。
- 新导入任务每次完全刷新导入任务列表，只保留新导入数据。
- 新导入任务不清空、不覆盖临时任务。
- 重新导入时，已存在并已有排期的临时任务作为固定占用时间保留，新导入任务绕开这些时间。
- 原排期在过去时间内的任务，在日历上显示划除。
- 已经开始或已经过去的原排期任务，在自动刷新排期时保持原排期，不被挪到新的日期。
- 如果已开始任务仍占用今天或后续工作日，新排入任务需要避让该任务。
- 新导入任务中如果存在与过去已划除临时任务相同的 SKU，新导入 SKU 正常排期，不影响过去的临时任务历史显示。

## 八、摄影排期规则

摄影导入任务排序：

1. 先排有预计到港时间的任务。
2. 有预计到港时间时，到港时间越临近越靠前。
3. 再排无预计到港时间但有样品的任务。
4. 样品字段非空视为有样品。
5. 无预计到港时间但有样品的任务可以随机/按创建顺序排。
6. `sku别名` 中数字相同的任务需要相邻排期。
7. 标灰任务排在临时任务之后。

摄影标灰规则：

- 有预计到港时间的摄影导入任务，高亮显示。
- 如果任务无预计到港时间且存在最早交期：
  - 当天时间早于 `最早交期 + 15天`，显示为浅灰色背景 + 白色字体。
  - 当天时间等于或晚于 `最早交期 + 15天`，正常显示。

摄影临时任务排序：

- 手动填写的临时任务优先排在标灰任务之前。
- 临时任务排在正常显示颜色任务之后。

## 九、摄像排期规则

摄像导入任务排序不同于摄影。

摄像可排期状态：

```text
待分配
待拍摄
拍摄中
```

规则：

1. `待分配`、`待拍摄`、`拍摄中` 均参与排期。
2. 其他状态不参与排期，并清空其排期。
3. 高优先级任务优先排期。
4. 创建时间早的任务排在前面。
5. 相同 SKU 的任务相邻排期。
6. 待分配任务没有摄像师时，统一归入虚拟人员 `待分配` 进行排期。
7. 摄像排期应连续排期，不应隔天排。
8. 导入新任务、更新任务时长、更新任务状态后自动刷新排期。
9. 摄像日历拖拽交互和相邻任务重排规则需与摄影保持一致。

摄像日历颜色规则：

- 常规任务正常颜色显示。
- 待分配任务显示灰色。
- 截至当天距离创建时间大于 1 个月的任务高亮显示。
- 临时任务高亮显示。
- 摄像任务日历卡片显示创建日期时，只显示年月日，不显示具体时间点。

## 十、任务列表规则

摄影/摄像导入任务列表：

- 支持分页。
- 单页最多 10 条。
- 支持搜索 `sku别名`。
- 搜索命中后：
  - 任务列表行高亮。
  - 对应日历任务卡片高亮定位。
  - 如果任务在其他月份，日历月份自动跳转到对应月份。
- 支持在列表中修改可调整字段。
- 导入任务的 `sku别名`、`产品标题`、`产品类型`、`最早交期` 为导入锁定字段，导入后只读显示，不支持手动修改。
- 摄像导入任务的 `sku别名`、`任务语种`、`任务类型`、`优先级`、`创建时间`、`关联sku` 为导入锁定字段，导入后只读显示，不支持手动修改。
- 摄像导入任务列表中的 `关联sku` 需要折叠显示：默认展示简短预览，点击后展开完整内容，再次点击可收起。
- 修改后自动保存并刷新排期。
- 支持单条导入任务删除。

临时任务列表：

- 支持编辑字段。
- 支持删除单条临时任务。
- 支持勾选已完成。
- 勾选已完成后：
  - 从临时任务列表消失。
  - 从活动排期中移除。
  - 写入已完成临时任务数据库。
  - 自动写入完成时间。

```text

对接人
任务备注
负责人

```

删除规则：

## 十二、个人产能看板

个人产能卡片展示：

```text
人员名称
总产能上限
剩余SKU
已排SKU
任务数
```

总产能上限：

- 每月每个人独立设置。
- 支持自定义输入。
- 输入一个数字后不应丢失焦点，不需要额外点击才能继续输入。
- 默认总产能上限按当月剩余工作日折算：
  - 摄影：2天 = 1个SKU
  - 摄像：1天 = 1个SKU

月份范围：

- 产能看板支持本月和次月切换。
- 点击个人卡片后进入个人本月/次月任务排期。
- 个人日历还支持通过拖拽边界进入上月。

### 摄影个人指标

摄影个人剩余 SKU：

```text
按本月剩余工作日扣除未标灰任务占用天数后折算，
同时不超过总产能上限扣除已排SKU后的余额；
摄影按 2天 = 1个SKU 计算。
```

摄影已排 SKU：

```text
统计本月已排进日历的摄影任务 SKU 数量，按 SKU 去重。
```

摄影任务数：

```text
统计本月参与摄影剩余产能计算的任务数量。
```

### 摄像个人指标

摄像个人剩余 SKU：

```text
按本月剩余工作日扣除未标灰任务占用天数后折算，
同时不超过总产能上限扣除已排SKU后的余额；
摄像按 1天 = 1个SKU 计算。
```

摄像已排 SKU：

```text
当月已排进日历的摄像高亮任务数量，不去重。
```

摄像任务数：

```text
当月摄像师所负责的所有任务数量，包含高亮和非高亮任务。
```

### 组剩余 SKU 指标

摄影组剩余 SKU：

```text
所有摄影师本月剩余SKU求和。
```

摄像组剩余 SKU：

```text
摄像师本月剩余SKU求和 - 待分配任务数。
```

注意：

- `待分配` 不作为真实摄像师参与摄像组剩余 SKU 求和。
- 摄像组剩余 SKU 最低显示为 `0 SKU`，不能显示负数。

## 十三、个人排期日历

个人日历入口：

```text
产能看板 -> 点击某个人员卡片
```

展示规则：

- 桌面端日历样式参考手机日历，但要适配桌面。
- 7列日历网格。
- 显示上月/本月/次月补位日期。
- 周末显示为休，不排期。
- 任务卡片显示：
  - SKU
  - 负责人/任务内容
  - 最早交期、创建日期、预计耗时、完成状态等辅助信息
- 过去任务在日历上划除。
- 已完成临时任务显示划除和已完成状态。

排期日历说明：

- 日历标题旁放问号图标。
- 鼠标停留问号时显示透明灰底浮窗。
- 说明内容：

```text
排期日历：展示当前人员在所选月份的任务排期；
可拖动任务到工作日调整日期，
拖到左上角/右下角边界可切换上月/下月；
调整后相邻任务会按移动任务的预计耗时自动前移或后延。
```

## 十四、拖拽排期规则

日历任务卡片可拖拽。

基础规则：

- 只能拖到工作日。
- 不能拖到周六、周日。
- 拖拽任务到目标日期后，任务从目标工作日开始排期。
- 任务结束日期按预计耗时自动计算。
- 摄影和摄像使用同一套拖拽排期逻辑。

相邻任务重排规则：

- 任务被拖到其他日期后，被占用日期的原任务自动后置。
- 原任务日期之后的任务自动往前排期，填充空出来的日期。
- 相邻任务提前/延后的天数以“被移动任务的预计耗时”为准。
- 同一人员的任务按时间顺序重排。
- 任务重排时保留双休规则。
- `0.5天` 任务按半天占用处理。

跨月拖拽规则：

- 拖到个人日历左上角第一个单元格/当月第一天边界时，自动切换到上个月。
- 拖到个人日历右下角最后一个单元格/当月最后一天边界时，自动切换到下个月。
- 一次拖拽过程中只能触发一次跨月跳转。
- 切换后用户可继续拖到目标日期完成落点。
- 上月/次月切换后仍需遵循工作日落点规则。

## 十五、导入后对原排期的影响

导入新的摄影/摄像任务后：

1. 对应板块的导入任务列表完全刷新。
2. 只保留本次新导入的导入任务。
3. 原有导入任务被替换。
4. 临时任务不受影响。
5. 已有排期的临时任务保持原日期。
6. 新导入任务自动重新排期。
7. 新导入任务会避让已固定的临时任务排期。
8. 自动排期按钮不需要存在。
9. 如果新导入任务与过去已划除临时任务 SKU 相同，新导入 SKU 正常排期，过去的临时任务仍保留在日历中作为划除历史记录。

一句话规则：


## 十八、核心算法伪代码

### 自动排期

```js
function autoSchedule(board, preserveTempSchedule = true) {
  if (board === "video") {
    for (task of imported.video) {
      if (!["待分配", "待拍摄", "拍摄中"].includes(task.taskStatus)) {
        task.start = "";
        task.end = "";
      }
    }
  }

  const people = unique(allTasks(board).map(taskSchedulePerson).filter(Boolean));
  for (person of people) {
    const cursor = firstWorkdayFromToday();
    const fixedTemps = preserveTempSchedule
      ? tempTasksForPersonWithExistingSchedule(person)
      : [];

    for (task of scheduleOrderForPerson(board, person)) {
      if (preserveTempSchedule && task.kind === "temp" && task.start && task.end) continue;
      updateDuration(task);
      scheduleTaskSkippingFixed(task, cursor, fixedTemps);
    }
  }
}
```

### 摄像排序

```js
function sortVideoTasks(tasks) {
  return groupBySku(tasks)
    .sortBy(priorityRank, earliestCreatedTime)
    .flatMap(group => group.sortBy(priorityRank, createdTime));
}
```

### 拖拽后重排

```js
function reflowPersonScheduleFromDrop(board, movedTask, targetDate) {
  const person = taskSchedulePerson(movedTask);
  const oldDate = movedTask.start;
  const otherTasks = tasksForPersonByDate(board, person, movedTask.id);

  if (targetDate > oldDate) {
    compactTasksBetweenOldAndTargetForward(otherTasks);
    placeMovedTaskAtTargetOrAfterGap(movedTask);
    compactTasksAfterMovedTask(otherTasks);
  } else {
    placeMovedTaskAtTarget(movedTask);
    compactTasksAfterTarget(otherTasks);
  }
}
```

### 摄像组剩余 SKU

```js
function videoGroupRemainingSku(month) {
  const assignedPeopleSku = videoPeople
    .filter(person => person !== "待分配")
    .sum(person => personCapacity(person, month).remainingSku);

  return Math.max(0, assignedPeopleSku - pendingAssignmentTaskCount(month));
}
```

## 十九、验收清单

另一台设备重建后，按以下清单验收：

- 摄影后台无日历看板。
- 摄像后台无日历看板。
- 摄影导入任务列表字段正确。
- 摄像导入任务列表字段正确且无图片字段。
- 导入任务列表每页最多 10 条。
- 搜索 SKU 可高亮列表和日历任务。
- 手动临时任务字段正确。
- 临时任务人员为单选：吕皇勇、汤崇武。
- 预计耗时支持 0.5 天。
- 导入后自动排期，无自动排期按钮。
- 重新导入只刷新导入任务，不影响临时任务。
- 已有临时任务排期在重新导入后保持不变。
- 摄影标灰规则正确。
- 摄像待分配任务参与排期。
- 摄像待分配任务灰色显示。
- 摄像创建时间超过 1 个月任务高亮。
- 摄像日历显示创建日期不显示具体时间点。
- 个人日历可拖拽任务调整日期。
- 拖拽到左上角只跳转一次上月。
- 拖拽到右下角只跳转一次下月。
- 拖拽后相邻任务按移动任务耗时前移/后延。
- 过去任务在日历上划除。
- 临时任务勾选完成后进入已完成数据库。
- 已完成数据库可按月份/季度查看。
- 删除已完成任务前有二次确认。
- 摄影组剩余 SKU 为摄影师剩余 SKU 求和。
- 摄像组剩余 SKU 为摄像师剩余 SKU 求和减待分配任务数。
- 摄像个人已排 SKU 统计当月高亮数量，不去重。
- 摄像个人任务数统计当月该摄像师全部任务。
- 指标问号悬停显示透明灰底规则说明。
- 排期日历问号悬停显示透明灰底规则说明。
